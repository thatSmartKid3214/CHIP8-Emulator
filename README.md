A Chip 8 emulator:
Only works for Windows.

Requires SDL3 to compile and run.

Python is required to build the project. You can build it using the Python file "build.py".

Ctrl + L => Load a ROM

Controls:
Keypad       Keyboard
+-+-+-+-+    +-+-+-+-+
|1|2|3|C|    |1|2|3|4|
+-+-+-+-+    +-+-+-+-+
|4|5|6|D|    |Q|W|E|R|
+-+-+-+-+ => +-+-+-+-+
|7|8|9|E|    |A|S|D|F|
+-+-+-+-+    +-+-+-+-+
|A|0|B|F|    |Z|X|C|V|
+-+-+-+-+    +-+-+-+-+

Issues:
- Runs slowly
- Sometimes fails to accurately emulate some ROMs.
